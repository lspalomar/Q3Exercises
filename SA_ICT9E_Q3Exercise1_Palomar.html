HTML

HyperText Markup Language

HTML is the standard language for web pages. In HTML you can creat your own website. It consists of a series of elements. It is the design you see in all websites.

CSS

Cascading Style Sheets

It is the design of the elements in a website or HTML. Can control the layout of multiple HTML elements and web pages. Also used to seperate style and content.

Bootstrap

Bootstrap Version 4

Free front end framework for faster and easier web development. Has the ability to easily create responsive designs. Can be used in all devices. Compatible in many different browsers.
